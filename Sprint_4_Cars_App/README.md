# Sprint 4: Car Sales Ads Project

## Project description
In this project, the first task was to perform Exploratory Data Analysis (data preprocessing, data visualizations, and hypothesis testing)  with a dataset on car sales advertisements. The other major task of this project was to develop and deploy a web application to the cloud service, Render, so that it is accessible to the public.

The link to the web app can be found at:
https://vehicles-sprint4-proj.onrender.com/
